> Design dark elegante, totalmente responsivo e com animações suaves.

🔗 **Live demo:** [gabrielmsm2.github.io/Projeto-Portfolio](https://gabrielmsm2.github.io/Projeto-Portfolio/)

---

## 👋 Sobre

Sou Gabriel Moreira Souto Mayor, estudante de **Ciência da Computação** no UniCEUB (Brasília — DF) com grande interesse em **segurança da informação**, redes e desenvolvimento de soluções. Este repositório hospeda meu portfólio pessoal, que reúne projetos, experiências profissionais e formas de contato.

---

## ✨ Destaques do projeto

- 🎨 **Design dark** com detalhes em verde neon e roxo — identidade visual voltada para cybersecurity
- 📱 **Totalmente responsivo** — funciona bem em desktop, tablet e mobile
- ⚡ **Animações ao scroll** — revelações suaves usando IntersectionObserver
- 📊 **Seção de skills** com barras de progresso animadas por categoria
- 🔗 **Navegação suave** entre seções com menu fixo
- 🌐 **Grid animado** de fundo com glow parallax
- 🎯 **Foco em performance** — HTML, CSS e JS vanilla, sem frameworks pesados

---

## 🛠️ Tecnologias

| Tecnologia | Uso |
|---|---|
| **HTML5** | Estrutura semântica |
| **CSS3** | Estilização, animações e layout responsivo (Grid + Flexbox) |
| **JavaScript (Vanilla)** | Interatividade, scroll reveal e menu mobile |
| **Google Fonts** | Space Grotesk (títulos) + JetBrains Mono (código) |
| **GitHub Pages** | Hospedagem |
