Projeto Focus
O projeto consiste no desenvolvimento de um aplicativo multiplataforma chamado “Focus”, criado para ajudar estudantes e profissionais a combater a procrastinação e melhorar a gestão de tempo, rotina e estudos. A proposta é reunir várias ferramentas de produtividade em um só lugar, funcionando como um verdadeiro canivete suíço da organização pessoal.

Além da versão mobile, o Focus também foi desenvolvido como um site (desenvolvimento web) utilizando HTML, CSS e JavaScript, oferecendo as mesmas funcionalidades de forma acessível pelo navegador.

Também desenvolvi uma versão do Focus como aplicativo utilizando Oracle APEX, integrado com HTML, CSS e JavaScript, permitindo uma interface mais robusta e conectada a banco de dados.

• Tecnologias utilizadas: HTML5, CSS3, JavaScript, Oracle APEX
• Funcionalidades: Cronômetro Pomodoro, lista de tarefas, controle de metas, estatísticas de produtividade, bloqueio de distrações e interface web responsiva.

Otimização de Código Python (Projeto Acadêmico)
Projeto desenvolvido para a disciplina de Organização e Arquitetura de Computadores no CEUB. O objetivo foi analisar e melhorar o desempenho de códigos Python, identificando partes críticas e reescrevendo trechos em linguagens de baixo nível, como C, C++ e Assembly, para aumentar a velocidade.

O estudo incluiu análise de desempenho, uso de ferramentas de profiling, testes, comparações de resultados e a aplicação de técnicas modernas de otimização. O trabalho demonstrou como reescrever pequenos trechos pode gerar melhorias reais de performance.

• Tecnologias e ferramentas: Python, C/C++, Assembly, Cython, Numba, PySpy, Scalene, Intel VTune
• Foco do projeto: otimização de desempenho, identificação de gargalos, reescrita de trechos críticos, análise arquitetural e validação de resultados.
